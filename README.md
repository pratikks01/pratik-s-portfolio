## Run Locally
Clone the project
```bash
  git clone https://github.com/pstricks01/pratik-s-portfolio.git
```
Go to the project directory

```bash
  cd Server
```

Install dependencies

```bash
  npm install
```
Start the server

```bash
  npm run start
```
